Деякі Фокуси підвищують Навичку (-1 до 0, 0 до +1…). Не можуть підняти Навичку вище ніж Рівень+1 при створенні персонажа. Якщо обраний Фокус підвищує Навичку понад це обмеження, оберіть іншу Навичку (крім Магії).

Після вибору Фокусів, вони, замість підвищення Навички по рівнях, надають 3 Одиниці Навичок для подальшого підвищення Навички до наступного рівня.

Після створення Фокуси можуть підвищувати Навички понад обмеження за Рівнем персонажа. Максимум Навички +4. Досягнувши +4, виберіть іншу Навичку для подальшого розвитку.

Фокус Воїна — будь-який, який схвалить Майстер. Фокус Експерта — будь-який, який схвалить Майстер.
