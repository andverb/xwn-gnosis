### Ініціатива

**Кидок**: 1к8 + мод. СПР або ІНТ.

Персонажі кидають ініціативу окремо, а супротивники використовують групову ініціативу. Супротивники не отримують бонусу до ініціативи, якщо вони не швидкі. У разі однакових результатів виграють персонажі.

### Бонус Атаки

Класовий бонус атаки + модифікатор Здібності (мод. здіб.) + навичка (-2, якщо навичка < 0) + Фокуси/Здібності/Магія/Модифікації зброї + модифікатори обставин

### Модифікатори для Атак

- Атака по сусідньому лежачому ворогу (дальня чи ближня): +2
- Половинне укриття: -2
- Атака за межами першого діапазону зброї: -2
- Стрільба по віддаленому лежачому ворогу: -2
- Майже повне укриття: -4
- Метання зброї, перебуваючи під атакою ближнього бою: -4
- Атака ближнього бою, перебуваючи лежачи: -4
- Раптова атака: -4
- Стрільба з лука, перебуваючи під атакою ближнього бою: неможливо

### Кидки Шкоди (ШКД)

Кубик ушкоджень від зброї + модифікатор Здібності + Фокуси + Навички + Магія/Модифікації зброї.

### Шокова Шкода

Шокова шкода зброї + модифікатор Здібності + Фокуси + Навички + бонус від Магічної/Модифікованої зброї.

- Коли виконується атака, захисник повідомляє свій Клас Броні (КБ), щоб нападник міг визначити, чи застосовуються шокові ушкодження.
- Якщо атака не влучила, але КБ цілі ≤ значення шокової шкоди зброї, вона завдає шокових ушкоджень.
- Шокова шкода метальної зброї застосовується лише до атак ближнього бою.
- Якщо атака влучила, отримана шкода не може бути меншою за показник шокової шкоди, якщо ціль сприйнятлива до шоку.

### Двозбройний Бій

Вимоги: Здібність Колюча зброя ≥ +1.

- Бонус до ушкоджень: +2 (не застосовується до шокових ушкоджень).
- Обмеження: -1 до влучання. Гравець обирає, якою зброєю завдається удар.
- Неможливо використовувати цей вид атаки без зброї в руках.
- Щити можна використовувати для двозбройних атак, але в цьому випадку вони не надають бонусу до Класу Броні (КБ) або захисту від шоку до наступного ходу персонажа.

### Верховий Бій

- Вершники без навички Їзда отримують -1 до влучання.
- Неможливо використовувати довгі луки, рушниці чи заклинання під час їзди.
- Їздові тварини використовують ініціативу свого вершника, але мають свої окремі дії (Основну та Руху), які можна чергувати з діями вершника.
- Посадка на тварину чи спішування вимагає Дії Руху.
- Треновані бойові тварини можуть атакувати, звичайні атакують лише для самозахисту, і вимагають перевірки Моралі або Інстинкту кожен раунд, інакше втікають із вершником.
- Вершник може змусити тварину підкорятися через перевірку Харизма/Їзда (Хар/Їзда) або Спритність/Їзда (Сприт/Їзда) як Дію Руху.
- Маунт із ПЗ ≤ 0 падає і скидає вершника.

### Спеціальні Бойові Маневри

Рекомендації для імпровізованих бойових дій:

- Якщо маневр передбачає удар по ворогу (будь-яким предметом чи зброєю), використовується кидок атаки для перевірки влучання.
- Якщо маневр вимагає маніпуляції об'єктами біля ворога, але не атакує його напряму, це перевірка навички або протиставлена перевірка (опосередковане змагання).

Результати успішного маневру:

- Ушкодження, завдане маневром, має бути ≥ базових ушкоджень зброї персонажа +2 або +4 (залежно від креативності).
- Якщо маневр спрямований більше на обмеження ворога, ніж на завдання ушкоджень, він може позбавити ворога Основної Дії або Дії Руху в його наступному ході.
- Якщо маневр поєднує завдання ушкоджень та обмеження, обидва ефекти накладаються частково.
